🎮 开始使用等级系统

第一次使用？请按以下步骤操作：

  1. 点击"➕ 创建新群组"按钮
  2. 输入您的Telegram群组ID（例如：-1002793326688）
  3. 设置群组名称（例如：上海小鸡管家群）
  4. 保存后即可在群组中使用 /level 命令

💡 如何获取群组ID？

  • 在群组中发送消息，查看Bot日志
  • 使用 @userinfobot 等机器人获取
  • 群组ID通常以 -100 开头

➕ 创建新群组配置

📋 配置说明

创建群组配置后，用户就可以在对应的Telegram群组中使用等级系统功能了。

重要：请输入您的Telegram群组ID（通常以-100开头的负数）
💡 您可以在群组中发送任意消息，然后查看Bot日志获取群组ID
用于在管理界面中识别群组，可以设置为任意友好的名称

🚀 创建后可以做什么？

  • 在群组中使用 /level 查看等级信息
  • 使用 /badges 查看勋章收集
  • 使用 /ranking 查看排行榜
  • 通过管理面板配置等级、积分、勋章等
